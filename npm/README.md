# LegacyBlazorJs

Rebuilds the official ASP.NET Core [blazor.web.js](https://github.com/dotnet/aspnetcore/tree/main/src/Components/Web.JS) for multiple JavaScript language targets.

The generated JavaScript files are published under the `dist/` directory of this package.

Please refer to the [GitHub repository](https://github.com/arika0093/LegacyBlazorJs) for details.
